    Quick Eats

    "Quick Eats" is a mobile web application built for a project in my Mobile Web Application course. The project was to design a product that uses a mobile web application that you create. For the application we created a restaurant randomizer that will display a restaurant and its information based off of your location that you enter. If you do not like or want to go to that restaurant then you simply hit next and it will display another random restaurant. If you however want to go to the restaurant then you can hit the directions button that will open Google Maps and give you directions. It is designed to be a simple quick solution to finding a restaurant if you can't make up your mind of where you want to go.

    I Heart Paint

    The "I Heart Paint" project was made for a coding competition sponsored by The University of Texas at Dallas and several other Texas universities. For the project we choose to use a mix of hardware and software to create a potential product that could be released. Using a Pebble smartwatch, an iLumi smar tbulb, an Android application, and a web application we created a tool for interior designers to help them choose paint colors from the smartwatch and display them on a wall using the smart bulb so they could get an idea of how it looked before buying it. When choosing a color they like they select it on the smartwatch and it gets sent through the android application to the web app giving them a color code they can use to buy the actual paint with online. "I Heart Paint" won best use of Pebble smartwatch, best use of iLumi smartbulb, and placed 11th overall.
